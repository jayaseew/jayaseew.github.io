If the caller isn't sure, get them to print something outside of landonline e.g Microsoft Word / MSPaint. If this is successful, get the user to try print a Fees Report within Landonline to ensure its not related to the specific document they're printing.

Get the user to close out of Uniprint via System Tray, and then relaunch it. The user does not need to close out of Landonline to do this, they can just minimise it for now.



Expanding System Tray: This is fairly easy, just ask the user to click the arrow symbol to the left of the icons in their system tray.



1) Get the user to right click the Blue UP icon in their system tray, located on the bottom right hand corner of their primary display, then select "Exit" (There should also be a Grey UP symbol as well as the Blue one, do not worry about the Grey one). If the user cannot see it, get them to expand the system tray by clicking the expand icon.





2) Once closed, get the user to click their start menu, and whilst expanded ask them to type "Uniprint". Ask them to then click "Uniprint Client".Nothing will pop up, but if you get the user to look on the bottom right hand corner of their screen, the Blue UP symbol should of reappeared. IF they cannot find Uniprint this way, try the alt method below.

ALT METHOD) Once closed, get the user to navigate to C:\Program Files\Uniprint Client\ and locate UPCinit.exe, double click it. Nothing will pop up, but if you get the user to look on the bottom right hand corner of their screen, the Blue UP symbol should of reappeared



3) Now get the user to try print again; in most cases this should of resolved the problem. If they still can't print, check the Citrix Permissions

Advise the user to speak with their IT as the printing issue isn't isolated to Landonline, so is highly likely an issue with the printer or printer settings.

If restarting Uniprint didn't resolve the issue, lets ensure the Citrix permissions are set correctly.

1) Right-click the Citrix Workspace icon in the system tray, 2) Select "Connection Center"


3) select LOLPRDORBXDA035 and then select "preferences" 4) ensure "Read and Write" is selected, then apply settings

You'll need to get the user to Uninstall/Reinstall Uniprint, restart the PC, and then try again.
